# AI Assistant Response Orb

Build a pulsing, animated orb that reacts to four states: **Idle**, **Listening**, **Thinking**, and **Responding**.

- Implement `Orb.tsx` (canvas-based orb), `OrbControls.tsx` (state buttons), and `useOrbState.ts` (animation params per state).
- Run: `npm install && npm run dev` — then `npm test` to run the test suite.
