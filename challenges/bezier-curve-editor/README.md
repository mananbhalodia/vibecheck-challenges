# Interactive Bézier Curve Editor

Build a visual editor for cubic Bézier curves with draggable control points, easing preview, and CSS export.

- Implement `BezierEditor`, `CurveCanvas`, `EasingPreview`, `PresetSelector`, `RecentCurves`, and `bezier.ts` / `coordinates.ts`.
- Run: `npm install && npm run dev` — then `npm test`.
